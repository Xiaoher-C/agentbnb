1. `initFreeTierTable(db: Database.Database): void` — Creates the tracking table:
   ```sql
   CREATE TABLE IF NOT EXISTS credit_free_tier_usage (
     agent_public_key TEXT NOT NULL,
     skill_id TEXT NOT NULL,
     usage_count INTEGER NOT NULL DEFAULT 0,
     last_used_at TEXT NOT NULL,
     PRIMARY KEY (agent_public_key, skill_id)
   )
   ```

2. `recordFreeTierUse(db: Database.Database, agentPublicKey: string, skillId: string): void` — Upserts usage count:
   ```sql
   INSERT INTO credit_free_tier_usage (agent_public_key, skill_id, usage_count, last_used_at)
   VALUES (?, ?, 1, ?)
   ON CONFLICT(agent_public_key, skill_id)
   DO UPDATE SET usage_count = usage_count + 1, last_used_at = ?
   ```

3. `getFreeTierUsage(db: Database.Database, agentPublicKey: string, skillId: string): number` — Returns current usage count (0 if no record).

Create `src/registry/free-tier.test.ts` testing all 5 behaviors using `openCreditDb(':memory:')`.
