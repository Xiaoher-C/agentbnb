  <name>Task 1: Hub Agent types, crypto, and SQLite store</name>
  <files>src/hub-agent/types.ts, src/hub-agent/crypto.ts, src/hub-agent/crypto.test.ts, src/hub-agent/store.ts, src/hub-agent/store.test.ts</files>
  <behavior>
    - encrypt("my-api-key", masterKey) returns a string containing iv:authTag:ciphertext (hex-encoded)
    - decrypt(encrypted, masterKey) returns the original plaintext
    - decrypt with wrong key throws an error
    - getMasterKey() reads HUB_MASTER_KEY env var (hex, 32 bytes); throws if missing or invalid length
    - createHubAgent(db, req) inserts a row into hub_agents with generated Ed25519 keypair, encrypted secrets, and returns HubAgent
    - getHubAgent(db, agentId) returns null for nonexistent agent
    - getHubAgent(db, agentId) returns HubAgent with decrypted secrets for existing agent
    - listHubAgents(db) returns array of HubAgent objects (secrets redacted — no plaintext API keys in list)
    - updateHubAgent(db, agentId, updates) modifies skill_routes and/or name; re-encrypts if secrets change
    - deleteHubAgent(db, agentId) removes the row; subsequent get returns null
    - initHubAgentTable(db) is idempotent — calling twice does not error
  </behavior>
  <action>
    Create `src/hub-agent/types.ts`:
    - Define `SkillRouteSchema` (Zod): `{ skill_id: string, mode: 'direct_api' | 'relay' | 'queue', config: ApiSkillConfig (for direct_api) | { relay_owner: string } (for relay) | {} (for queue) }`
    - Define `HubAgentSchema` (Zod): `{ agent_id: string, name: string, owner_public_key: string, public_key: string, skill_routes: SkillRoute[], status: 'active' | 'paused', created_at: string, updated_at: string }`
    - Define `CreateAgentRequestSchema` (Zod): `{ name: string, skill_routes: SkillRoute[], secrets?: Record<string, string> }`
    - Export all types and schemas

    Create `src/hub-agent/crypto.ts`:
    - `getMasterKey()`: Read `HUB_MASTER_KEY` from process.env (64-char hex = 32 bytes). Throw if missing or wrong length.
    - `encrypt(plaintext: string, masterKey: Buffer): string` — AES-256-GCM. Generate random 12-byte IV. Return `${iv.toString('hex')}:${authTag.toString('hex')}:${ciphertext.toString('hex')}`.
    - `decrypt(encrypted: string, masterKey: Buffer): string` — Split on ':', decode hex, decrypt, return plaintext.

    Create `src/hub-agent/store.ts`:
    - `initHubAgentTable(db)`: CREATE TABLE IF NOT EXISTS hub_agents (agent_id TEXT PRIMARY KEY, name TEXT NOT NULL, owner_public_key TEXT NOT NULL, public_key TEXT NOT NULL, private_key_enc TEXT NOT NULL, secrets_enc TEXT, skill_routes TEXT NOT NULL DEFAULT '[]', status TEXT NOT NULL DEFAULT 'active', created_at TEXT NOT NULL, updated_at TEXT NOT NULL). Uses `getMasterKey()` for encrypt/decrypt.
    - `createHubAgent(db, req, ownerPublicKey)`: Generate Ed25519 keypair via `generateKeyPair()`. Derive agent_id via `deriveAgentId()`. Encrypt private key and secrets with master key. Insert row. Bootstrap 50 credits via `bootstrapAgent(creditDb, agentId, 50)` (creditDb passed as param). Return HubAgent.
    - `getHubAgent(db, agentId, masterKey)`: SELECT, parse JSON skill_routes, decrypt secrets. Return HubAgent | null.
    - `listHubAgents(db)`: SELECT all, parse skill_routes. Do NOT decrypt secrets (list view). Return HubAgent[] with secrets omitted.
    - `updateHubAgent(db, agentId, updates, masterKey)`: UPDATE name, skill_routes, secrets (re-encrypt if provided). Return updated HubAgent.
    - `deleteHubAgent(db, agentId)`: DELETE FROM hub_agents WHERE agent_id = ?. Return boolean (true if row deleted).
  </action>

    - PUT /api/agents/:id with { name, skill_routes } returns updated agent

    - Export `hubAgentRoutesPlugin` as an async Fastify plugin (same pattern as creditRoutesPlugin).
    - Options: `{ registryDb: Database.Database, creditDb: Database.Database }`.
    - On plugin init: call `initHubAgentTable(registryDb)`.
    - All routes tagged `['hub-agents']` for Swagger grouping.
    - All routes use Fastify JSON Schema for request/response validation.

    Routes:
    1. `POST /api/agents` — Parse body as CreateAgentRequestSchema. Call `createHubAgent()`. After creation, build a v2.0 CapabilityCard from the agent's skill_routes and insert into registry via raw SQL (same pattern as relay upsertCard in Phase 30 — use AnyCardSchema + raw SQL, NOT store.ts insertCard which is locked to v1.0). Grant 50 credits via `bootstrapAgent(creditDb, agentId, 50)`. Return 201 with agent data (strip private_key and secrets from response).
    2. `GET /api/agents` — Call `listHubAgents()`. Return `{ agents: [...] }`.
    3. `GET /api/agents/:id` — Call `getHubAgent()`. If null, return 404. Else return agent (strip secrets and private_key from response — only return secret key names, not values).
    4. `PUT /api/agents/:id` — Parse body. Call `updateHubAgent()`. If agent not found, return 404. Also update the agent's CapabilityCard in registry to reflect new skill_routes.
    5. `DELETE /api/agents/:id` — Call `deleteHubAgent()`. Also delete the agent's CapabilityCard from registry. Return `{ ok: true }` or 404.
