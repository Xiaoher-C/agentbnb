  <behavior>
    - hold(owner, amount, cardId) returns { escrowId: string } after deducting credits
    - hold throws INSUFFICIENT_CREDITS when balance is too low
    - settle(escrowId, recipientOwner) transfers held credits to recipient
    - release(escrowId) refunds held credits to original owner
    - getBalance(owner) returns current credit balance (0 if unknown)
    - getHistory(owner, limit?) returns CreditTransaction[] newest-first
    - grant(owner, amount) bootstraps agent credits idempotently
    - All methods are async (returning Promises) for compatibility with future HTTP implementation
  </behavior>
  <action>
    1. Create `src/credit/credit-ledger.ts`:
       - Define `EscrowResult` type: `{ escrowId: string }`
       - Define `CreditLedger` interface with all 6 methods as async:

         - `settle(escrowId: string, recipientOwner: string): Promise<void>`
         - `release(escrowId: string): Promise<void>`
         - `getBalance(owner: string): Promise<number>`

         - `grant(owner: string, amount?: number): Promise<void>`
