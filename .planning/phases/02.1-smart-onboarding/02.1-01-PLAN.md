  <name>Task 1: Onboarding Detection and Card Generation</name>
  <files>src/cli/onboarding.ts, src/cli/onboarding.test.ts</files>
  <behavior>
    ## detectApiKeys(knownKeys)
    - Given env has OPENAI_API_KEY set, returns ['OPENAI_API_KEY']
    - Given env has no known keys, returns []
    - Given env has multiple known keys, returns all matching
    - SECURITY: Source code of onboarding.ts must NOT contain `process.env[` index access (only `key in process.env`)

    ## isPortOpen(port, host, timeoutMs)
    - Given port with listener, returns true
    - Given port with no listener, returns false
    - Given timeout exceeded, returns false

    ## detectOpenPorts(ports)
    - Returns array of ports that are open
    - Probes all ports in parallel (Promise.all)
    - Uses 300ms timeout per port

    ## buildDraftCard(apiKey, owner)
    - Given 'OPENAI_API_KEY' and owner 'alice', returns valid CapabilityCard with owner='alice', level=1, name containing 'OpenAI'
    - Given 'ELEVENLABS_API_KEY', returns card with apis_used=['elevenlabs']
    - Given unknown key 'RANDOM_KEY', returns null
    - Generated card validates against CapabilityCardSchema
    - Card has unique UUID id, spec_version '1.0', online=true

    ## KNOWN_API_KEYS constant
    - Contains at least: OPENAI_API_KEY, ANTHROPIC_API_KEY, ELEVENLABS_API_KEY, KLING_API_KEY, STABILITY_API_KEY, REPLICATE_API_TOKEN

    ## API_TEMPLATES constant
    - Has entry for each key in KNOWN_API_KEYS
    - Each template has name, description, level, inputs, outputs, pricing, metadata with apis_used and tags
  </behavior>
  <action>
    Create `src/cli/onboarding.ts` with:

Include templates for: OPENAI_API_KEY, ANTHROPIC_API_KEY, ELEVENLABS_API_KEY, KLING_API_KEY, STABILITY_API_KEY, REPLICATE_API_TOKEN, GOOGLE_API_KEY, AZURE_OPENAI_API_KEY, COHERE_API_KEY, MISTRAL_API_KEY.
    3. `detectApiKeys(knownKeys: readonly string[]): string[]` -- uses `key in process.env` (NEVER `process.env[key]`). Add a JSDoc security comment.
    4. `isPortOpen(port: number, host?: string, timeoutMs?: number): Promise<boolean>` -- TCP connect probe using `node:net` createConnection with 300ms default timeout.
    5. `detectOpenPorts(ports: readonly number[]): Promise<number[]>` -- Promise.all over isPortOpen.
    6. `buildDraftCard(apiKey: string, owner: string): CapabilityCard | null` -- looks up template, returns null if not found. Uses `randomUUID()` for id, sets spec_version '1.0', availability.online=true, timestamps.
