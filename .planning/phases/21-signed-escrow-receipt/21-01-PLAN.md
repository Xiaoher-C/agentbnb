<objective>
Create Ed25519 keypair generation on `agentbnb init` and implement the signed escrow receipt protocol types and functions.

Purpose: Enable agents to cryptographically prove credit commitment. This is the foundation for cross-machine credit verification — without signed receipts, providers cannot trust that a remote requester has committed credits.

Output: signing.ts (keypair + sign/verify), escrow-receipt.ts (receipt creation), updated CLI init, updated config types.
</objective>

    - `signEscrowReceipt(data: Record<string, unknown>, privateKey: Buffer): string`: JSON.stringify the data with sorted keys, sign with Ed25519, return base64url encoded signature.
    - `verifyEscrowReceipt(data: Record<string, unknown>, signature: string, publicKey: Buffer): boolean`: Reconstruct canonical JSON, verify Ed25519 signature. Return false (not throw) on invalid.
