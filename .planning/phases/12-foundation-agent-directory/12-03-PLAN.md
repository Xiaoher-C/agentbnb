1. Add to `hub/src/types.ts` (append at bottom, after AppOutletContext):
```typescript
/** Agent profile as returned by GET /api/agents */
export interface AgentProfile {
  owner: string;
  skill_count: number;
  success_rate: number | null;
  total_earned: number;
  member_since: string;
}

/** Activity entry as returned by GET /api/agents/:owner recent_activity */
export interface ActivityEntry {
  id: string;
  card_name: string;
  requester: string;
  status: 'success' | 'failure' | 'timeout';
  credits_charged: number;
  created_at: string;
}

/** Full agent profile response from GET /api/agents/:owner */
export interface AgentProfileResponse {
  profile: AgentProfile;
  skills: HubCard[];
  recent_activity: ActivityEntry[];
}
```

2. Create `hub/src/hooks/useAgents.ts` with TWO exported hooks:

**useAgents()** — fetches the agent list, follows the established 30s polling pattern:
   - State: `agents: AgentProfile[]`, `loading: boolean`, `error: string | null`
   - `isFirstFetch` ref pattern (same as useCards)
   - `fetchAgents` useCallback: `fetch('/api/agents')` → parse as `{ items: AgentProfile[]; total: number }` → setAgents(data.items)
   - Two useEffect calls: initial fetch + 30s interval
   - Returns `{ agents, loading, error }`

**useAgentProfile(owner: string)** — fetches a single agent profile:
   - State: `profile: AgentProfile | null`, `skills: HubCard[]`, `recentActivity: ActivityEntry[]`, `loading: boolean`, `error: string | null`
   - `isFirstFetch` ref pattern
   - `fetchProfile` useCallback with `owner` dependency: `fetch(\`/api/agents/${encodeURIComponent(owner)}\`)` → parse as `AgentProfileResponse`
   - Handle 404 response: set error to "Agent not found"
   - Two useEffect calls: initial fetch + 30s interval (re-fetch when owner changes)
   - Returns `{ profile, skills, recentActivity, loading, error }`

Both hooks follow the exact useCards pattern: isFirstFetch ref, loading only set false on first fetch, error cleared on success, existing data kept on subsequent fetch errors (graceful degradation).
