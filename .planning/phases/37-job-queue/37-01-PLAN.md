<objective>
Build a SQLite-backed job queue for Hub Agent offline requests and a relay bridge that auto-dispatches queued jobs when session agents reconnect.

Purpose: When a Hub Agent skill is routed to a session-based agent via relay and that agent is offline, the request currently fails immediately. This phase queues those requests and auto-dispatches them on reconnect, completing the Hub Agent relay/queue execution modes.

Output: job-queue.ts (SQLite CRUD), relay-bridge.ts (auto-dispatch on reconnect), updated executor.ts (relay+queue modes), updated routes.ts (job status endpoints), updated websocket-relay.ts (onAgentOnline hook).
</objective>
