  <name>Task 2: BudgetController — pre-calculate cost and enforce spending limits</name>
  <files>src/conductor/budget-controller.ts, src/conductor/budget-controller.test.ts</files>
  <behavior>
    - calculateBudget with 3 matched sub-tasks returns ExecutionBudget with correct estimated_total (sum of credits + 5 cr orchestration fee)
    - calculateBudget with max_budget=100 and estimated_total=50 sets requires_approval=false
    - calculateBudget with max_budget=30 and estimated_total=50 sets requires_approval=true
    - canExecute returns true when BudgetManager.canSpend(estimated_total) is true AND requires_approval is false
    - canExecute returns false when balance is below reserve + estimated_total
    - canExecute returns false when requires_approval is true (needs explicit approval)
    - ORCHESTRATION_FEE constant is 5
  </behavior>
  <action>
    1. Create `src/conductor/budget-controller.ts`:
       - Import `BudgetManager` from `../credit/budget.js`
       - Import `MatchResult`, `ExecutionBudget` from `./types.js`

       - Export `ORCHESTRATION_FEE = 5` (constant)

       - Export class `BudgetController`:
         - Constructor: `(budgetManager: BudgetManager, maxBudget: number)`
         - `calculateBudget(matches: MatchResult[]): ExecutionBudget`:
           a. Sum all match credits → sub_total
           b. estimated_total = sub_total + ORCHESTRATION_FEE
           c. per_task_spending = Map from subtask_id → credits
           d. requires_approval = estimated_total > maxBudget
           e. Return ExecutionBudget { estimated_total, max_budget: this.maxBudget, orchestration_fee: ORCHESTRATION_FEE, per_task_spending, requires_approval }
         - `canExecute(budget: ExecutionBudget): boolean`:
           a. If budget.requires_approval, return false (caller must get explicit approval first)
           b. Return this.budgetManager.canSpend(budget.estimated_total)
         - `approveAndCheck(budget: ExecutionBudget): boolean`:
           a. Ignores requires_approval flag (used after user explicitly approves)
           b. Return this.budgetManager.canSpend(budget.estimated_total)

       - Test: requires_approval=false when max_budget=100, estimated_total=35
       - Test: requires_approval=true when max_budget=20, estimated_total=35
       - Test: canExecute returns true when budget fits and no approval needed
       - Test: canExecute returns false when requires_approval=true
       - Test: canExecute returns false when balance insufficient (give agent only 30cr, reserve=20, available=10, estimated=35)
       - Test: approveAndCheck returns true when balance sufficient (ignores approval flag)
