1. **hub/src/hooks/useCards.ts** — Custom React hook for data fetching:

   ```typescript
   export function useCards()
   ```

   State managed by the hook:
   - `cards: HubCard[]` — current card list
   - `total: number` — total count from API
   - `loading: boolean` — true during initial fetch
   - `error: string | null` — error message if fetch fails
   - `query: string` — search query text
   - `level: number | null` — level filter (1, 2, 3, or null for all)
   - `category: string | null` — category filter
   - `onlineOnly: boolean` — online filter toggle

   Fetch logic:
   - Build URL: `/cards?` with query params: q (if non-empty), level (if set), online=true (if onlineOnly), limit=100
   - Use fetch() with no explicit host (relative URL works via Vite proxy in dev, same-origin in production)
   - On success: parse CardsResponse, set cards/total, clear error
   - On failure: set error message, keep existing cards if any (graceful degradation)

   Polling:
   - Use setInterval at 30000ms (30s) to re-fetch
   - Clear interval on unmount
   - Also re-fetch when query/level/onlineOnly changes (use useEffect with these as deps)

   Category filtering (client-side):
   - The /cards API doesn't have a category param, so filter client-side:
   - After fetching, if `category` is set, filter cards where inferCategories(card.metadata).categories includes a category with matching id
   - Compute `availableCategories` from ALL fetched cards (before client-side category filter) -- deduplicate category ids

   Return:
   ```typescript
   {
     cards, total, loading, error,
     query, setQuery, level, setLevel,
     category, setCategory, onlineOnly, setOnlineOnly,
     availableCategories, retry
   }
   ```

   `retry()` — manually re-fetch (for error state retry button)

   Stats computation:
   - `agentsOnline`: count unique owners where card.availability.online === true
   - `totalCapabilities`: total from API response
   - `totalExchanges`: 0 (no exchange tracking endpoint yet -- show 0 honestly per user decision "even with zeros")

   Return these stats too: `{ agentsOnline, totalCapabilities, totalExchanges }`
