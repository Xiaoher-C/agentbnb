### Task 2: Category + Utility Modules (TDD)

**hub/src/lib/categories.ts**
- `CATEGORY_MAP`: 10 known API names → category (openai/anthropic/azure-openai/google/cohere/mistral → text_gen; elevenlabs → tts; kling → video_gen; stability → image_gen; replicate → compute)
- `TAG_CATEGORY_MAP`: 30+ tag keywords → 14 category IDs
- `ALL_CATEGORIES`: Full 15-category definitions with lucide icon names
- `inferCategories(metadata?)`: API-first → tag-fallback → dedup by id → max 4 chips → custom default
- `getCategoryIcon(categoryId)`: Direct lookup, returns 'Puzzle' for unknowns

**hub/src/lib/utils.ts**
- `getLevelBadge(level)`: L1=Atomic/dot, L2=Pipeline/connected, L3=Environment/block
- `getStatusIndicator(online)`: true → 'emerald', false → 'rose' (two-state)
- `formatCredits(pricing)`: "5 credits" or "5-120 credits"
